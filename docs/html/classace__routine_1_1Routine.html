◆ run()

virtual int ace_routine::Routine::run ( )
pure virtual

The body of the routine.

The return value is never used. It exists solely to prevent the various ROUTINE_YIELD(), ROUTINE_DELAY() and ROUTINE_END() macros from accidentally compiling inside a nested method.

Returns
The return value is always ignored. This method is declared to return an int to prevent the user from accidentally returning from this method incorrectly. This method should always return using one of the ROUTINE_YIELD(), ROUTINE_DELAY() or ROUTINE_END() macros, or not return at all is using the ROUTINE_LOOP() macro.
