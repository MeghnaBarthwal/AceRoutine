static void ace_routine::RoutineScheduler::loop ( )
inlinestatic

Run the current routine using the current scheduler.

This method returns when the underlying Routine suspends execution, which allows the system loop() to return to do systems processing, such as WiFi. Everyone must cooperate to make the whole thing work.
